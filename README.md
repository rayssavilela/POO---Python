# 📘 Exercícios de Programação Orientada a Objetos em Python

Este repositório contém uma coleção de exercícios desenvolvidos durante meus estudos de **Programação Orientada a Objetos (POO)** utilizando Python, juntamente com exercícios de **Algoritmos** e **Estrutura de Dados**.
O foco é aplicar na prática os principais conceitos do paradigma orientado a objetos.

---

## 🧠 Conceitos abordados

✔️ Classes e Objetos
✔️ Encapsulamento
✔️ Herança
✔️ Polimorfismo
✔️ Métodos especiais (`__init__`, `__str__`)
✔️ Métodos de classe e estáticos
✔️ Listas
✔️ Tuplas

---

## 💻 Tecnologias utilizadas

* Python 3.10
* Paradigma Orientado a Objetos

---

## 🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de:

* Praticar lógica de programação
* Consolidar conceitos de POO
* Melhorar a organização e reutilização de código

---

## 🚀 Como executar

Clone o repositório:

```bash
git clone https://github.com/rayssavilela/POO---Python.git
```

Acesse a pasta:

```bash
cd POO---Python
```

Execute um arquivo:

```bash
python app.py
```
